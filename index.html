The Last Resort

This is a basic webpage that was made to practice the concepts of creating branches using Git. In it, myself and my partner were tasked with creating two seperate layouts for a resort's website, and then merging the preferred branch of the two into the master branch. What we ended up making was more of a 'worst vacation you could imagine' page, hence the name "The Last Resort".

Languages Used:
  1. HTML
  2. CSS
  3. Git/GitHub
Cupcake Shop

This is a website designed to show off 'Drew's Tasty Cakes', an imaginary Cupcake company. It was made to practice using specific tags to style certain elements on the page. I created a seperate page that contains a menu and pricing for several different cupcakes, and highlights which of the cakes is the best seller.

Languages Used:
  1. HTML
  2. CSS
  3. Git/GitHub
Maldives Vacation

Ever dream of white sand, gentle waves, and palm trees? Then look no further than this page, designed to highlight the Maldives. It gives a little history, and provides several travel packages to choose from, including pricing and accomodations.
